# MiniMind Datasets

将所有下载的数据集文件放置到当前目录.

Place the downloaded dataset file in the current directory.

## Notes

- Dataset files are large and not tracked by git (see `.gitignore`)
- Make sure you have enough disk space before downloading (~20GB+ for full dataset)
- Recommended to download datasets to an external drive if space is limited

## My Dataset Sources

- Pretrain data: [HuggingFace - minimind-v1-small](https://huggingface.co/datasets/jingyaogong/minimind-v1-small)
- SFT data: downloaded via the scripts in `scripts/` directory
- Note to self: last downloaded 2024-11, stored on external SSD under `~/datasets/minimind/`
